*FFT中的重点概念
采样时间长度：为采样总时间
采样频率：一秒均匀采集的信号点数(要大于采集信号中最高频率的两倍)
频率间隔：即最小频率单位，为采样频率/采样点数N  或者w0 = 1 / ( Tend - Tstart ). Tend为采样的结束时间;Tstart为采样的其实时间，一般取0
因为采样频率Fs = 采样点数N / 采样的总时间(Tend - Tstart)
而频率间隔w0 = 采样频率Fs / 采样点数N
综合上面两个式子可得：w0 = 1 / 采样的总时间(Tend - Tstart)  与  w0 = 采样频率Fs / 采样点数N  等价

个别文献或程序中w0 = 采样频率 / (采样点数N-1)  与  w0 = 采样频率Fs / 采样点数N  没有太大区别
